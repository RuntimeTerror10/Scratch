Search bar

Search bar is a location on a web page that allows visitors to search the site.

Add your own colors and placeholder text to create a search bar for your webpage.

Rectangular search bar

Add your own custom CSS classes in the code given to alter all the atributes of the search bar.

<div class="search_wrap rect-search">

<div class="search_box">

<input type="text"class="input" placeholder="search...">

<div class="btn btn_icon">

<i class="fas fa-search search-icon"></i>

</div>

</div>

</div>
